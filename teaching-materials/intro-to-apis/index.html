What is an API?

  • (A)pplication (P)rogramming (I)nterface
  • A transmitter of data in a universal format
  • A set of rules that determines how one application talks to another
    • The two applications can be separate websites, or the client and server of the same website

How do APIs work?


  • One application makes a request, and the other sends a response
  • When working with a REST API, common types of requests are GET, POST, & PUT
  • The Dev Tools Network tab helps us see requests that come and go on a webpage

How do APIs work?


  • The data in a request/response is in JavaScript Object Notation (JSON) format
  • Browser extensions like JSONView help humans look at JSON
  • See this example

How do APIs work?


  • We can get back different sets of data if we change what we are asking for in the request url
  • The API's docs should tell us how to format requests
  • Sometimes it's as simple as changing url routes
    • https://api.github.com/users/sashak007/repos
    • https://api.github.com/users
  • Sometimes it's changing query parameters
    • http://api.giphy.com/v1/gifs/search?q=dog&limit=10&api_key=dc6zaTOxFJmzC
  • Valid query parameters vary. Always check the API's docs to see what parameters are available for use

How can we use APIs?


  • We can use AJAX to make a request
  • See an example of XMLHttpRequest()

How can we use APIs?

In this class, we'll use jQuery's $.ajax() method


$.ajax({
    url: 'https://api.giphy.com/v1/gifs/search',
    data: {
        q: 'dogs',
        api_key: 'dc6zaTOxFJmzC',
        limit: 10
    },
    success: function(response){
        console.log(response);
    },
    fail: function() {
        console.log('Something went wrong!');
    }
});

How can we display the response?



$.ajax({
    url: 'https://api.giphy.com/v1/gifs/search',
    data: {
        q: 'dogs',
        api_key: 'dc6zaTOxFJmzC',
        limit: 10
    },
    success: function(response){
        var imgUrl = response.data[0].images.fixed_height.url;
        var imgTag = "<img src=" + imgUrl + ">";
        $(".img-container").append(imgTag);
    },
    fail: function() {
        console.log('There was a problem with the request response.');
    }
});

Let's Develop It


  • Update your success callback to display the first image on the page
  • Refactor your success callback code into a function that lives outside the ajax request
  • Use a loop to display all images to the page
  • BONUS: Make each image a clickable link
  • BONUS: Read through the Giphy API Docs to see what other query parameters we can include in our requests
